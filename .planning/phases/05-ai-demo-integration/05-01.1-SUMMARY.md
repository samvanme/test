## Decisions Made

- **Scrollbar approach**: Inline styles for Firefox (scrollbarWidth/scrollbarColor), embedded CSS for WebKit pseudoselectors - provides cross-browser support without external CSS
- **Chat anchor method**: flex-col-reverse automatically anchors new content at scroll position 0, eliminating need for manual scrollIntoView
- **Icon placement**: Defined as module-level components to satisfy react-hooks/exhaustive-deps lint rules
- **Rotation behavior**: Sequential rotation (not random) so users see different scenarios on replay without unpredictability
